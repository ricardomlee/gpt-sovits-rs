# gpt-sovits-rs 1.2.0

This release turns multi-voice inference into a bounded, observable local service. One HTTP process
can serve multiple fine-tuned GPT-SoVITS v2/v2Pro voices while keeping GPU work serialized and
memory use predictable.

## Install

Docker remains the recommended path. Images do not include model weights.

```bash
git clone https://github.com/ricardomlee/gpt-sovits-rs.git
cd gpt-sovits-rs
cp .env.example .env
docker compose -f compose.cuda.yml up -d
```

RTX 40 users should select:

```env
CUDA_IMAGE_TAG=1.2.0-cuda-sm89
```

## Highlights

- Added per-voice `gpt_model`, `sovits_model`, and `bigvgan_model` bindings in `voice.json`.
- Made CLI and HTTP voice selection honor the same model and v2Pro SV bindings.
- Added a bounded LRU model-pipeline cache with shared BERT/HuBERT resources.
- Kept all GPU model loading, warmup, and inference globally serialized.
- Added manual `POST /warmup` and startup `PRELOAD_VOICES` support.
- Added `GET /status` with cache hits, misses, evictions, queue depth, and busy state.
- Added request text, batch-size, and inference-queue limits with explicit HTTP errors.
- Restricted request-supplied reference and SV paths to `VOICES_DIR` by default.
- Bounded reference-feature caches for stable long-running memory use.
- Split the HTTP server into focused handler, lifecycle, state, request, response, and registry
  modules.

## Voice Model Binding

Relative model paths are resolved from `MODELS_DIR`; reference audio and SV paths remain relative
to the voice directory.

```json
{
  "reference_audio": "ref.wav",
  "reference_text": "参考音频对应的文字",
  "sv_embedding": "ref_sv.safetensors",
  "gpt_model": "carol/gpt.safetensors",
  "sovits_model": "carol/sovits.safetensors",
  "language": "zh"
}
```

## Service Controls

```env
MAX_CACHED_PIPELINES=2
MAX_TEXT_CHARS=10000
MAX_BATCH_ITEMS=64
QUEUE_TIMEOUT_SECS=120
PRELOAD_VOICES=diana,carol
```

Keep `PRELOAD_VOICES` within `MAX_CACHED_PIPELINES`. Requests that time out before entering the
serialized inference slot receive HTTP 503; active model loading and inference are not cancelled.

## Validation

Release validation included:

- Formatting, Clippy, full HTTP tests, and CUDA feature compilation.
- CPU and CUDA Compose configuration validation.
- Real v2Pro model smoke using voice-bound GPT/SoVITS weights and SV embedding.
- CUDA quality matrices for Diana, Carol, and Sun across short, normal, and long Chinese text.
- Nine generated clips with no waveform quality findings, clipping, or non-finite samples.
- Long-text inference of 2.60-2.80 seconds for 19.7-22.8 seconds of generated audio on an RTX
  4060 Ti.
