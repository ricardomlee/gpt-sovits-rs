# gpt-sovits-rs 1.1.0

Feature release focused on native GPT-SoVITS v2Pro inference, Python-free model conversion, and
complete binary/container delivery.

## Install

Docker remains the recommended path. The images include both `gpt-sovits` and
`gpt-sovits-convert`, but do not include model weights:

```bash
git clone https://github.com/ricardomlee/gpt-sovits-rs.git
cd gpt-sovits-rs

mkdir -p models/bert models/hubert voices outputs
gpt-sovits-convert gpt /path/to/s1bert25hz.ckpt models/gpt-model.safetensors
gpt-sovits-convert sovits /path/to/s2G2333k.pth models/sovits-model.safetensors
gpt-sovits-convert bert /path/to/chinese-roberta-wwm-ext-large/pytorch_model.bin models/bert/bert.safetensors
cp /path/to/chinese-roberta-wwm-ext-large/tokenizer.json models/bert/tokenizer.json
gpt-sovits-convert hubert /path/to/chinese-hubert-base/pytorch_model.bin models/hubert/hubert.safetensors

cp .env.example .env
docker compose -f compose.cpu.yml up -d
```

CUDA users should set `CUDA_IMAGE_TAG` in `.env` first:

```text
latest-cuda-sm75  RTX 20
latest-cuda-sm86  RTX 30
latest-cuda-sm89  RTX 40
latest-cuda-sm90  H100
```

Then run:

```bash
docker compose -f compose.cuda.yml up -d
```

## Highlights

- Added native v2Pro SoVITS inference, including `ge_to512`, `sv_emb`, and `prelu` support.
- Added optional speaker-verification embedding support through CLI, HTTP API, and voice profiles.
- Added `gpt-sovits-convert`, a Rust converter for GPT, SoVITS, Chinese RoBERTa, Chinese HuBERT,
  and v2Pro SV embedding files.
- Removed Python model-preparation scripts from the current runtime/conversion workflow.
- Included `gpt-sovits-convert` in release archives and CPU/CUDA Docker images.
- Kept model distribution explicit: release assets and Docker images do not bundle model weights.

## Model Preparation

Users should download official source checkpoints, reuse an existing GPT-SoVITS installation, or
train their own voices. This project does not mirror or redistribute model weights.

```bash
gpt-sovits-convert gpt /path/to/custom-gpt.ckpt models/gpt-model.safetensors
gpt-sovits-convert sovits /path/to/custom-sovits.pth models/sovits-model.safetensors
```

For v2Pro voices, convert the optional SV embedding generated by GPT-SoVITS preprocessing:

```bash
gpt-sovits-convert sv /path/to/logs/voice_v2pro/7-sv_cn/ref.wav.pt voices/demo/ref_sv.safetensors
```

See `docs/MODELS.md` for full model layout and license notes.
